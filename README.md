# Simple Blood Bank
Project for CSE435(Software Quality Assurance)

### Project Features
  * Written in ADO.NET (C#)
  * MSSQL Database
  * MVC Pattern (Partially still learning)
  * Unit Testing
    * nunit3
    * mstest

Application is ready to use.

#### A simple blood bank management system

### Application Features
  * Donor Registration
  * Manage Donor
  * Donation
  * Recipition
  * Donation Log
  * Reciption Log
  * Storage Infomation
